
reactris 2 - electric boogaloo
========

Simplified React tetris demo rendered to the DOM

[Play Demo](http://jmorrell.github.io/reactris/)

This should be a conflicting change

### Controls

| Key   | Action |
| ----- | ------ |
| Up    | Rotate |
| Down  | Down   |
| Right | Right  |
| Left  | Left   |
| Space | Drop   |

### To Run

Open `index.html`

### To Build

```
>> npm install
>> npm run build
```
